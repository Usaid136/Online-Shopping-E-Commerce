🛍️ Online Shopping Website – PHP MySQL Project

A fully responsive e-commerce website built using PHP, MySQL, and Bootstrap. It provides users with a seamless shopping experience and allows administrators to manage products and customer orders.

This project showcases a clean front-end layout with features like cart view, login/register system, and a dynamic product showcase section.
